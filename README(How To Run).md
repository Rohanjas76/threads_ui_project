# threads_ui_project
This project is a community-based discussion platform that allows users to share text posts across different topic groups. It features community navigation, popular tags, and interactive actions like liking, commenting, and sharing. The system focuses on clean design, responsiveness, and user engagement using modern web technologies.


# Community Discussion Platform

## Requirements
- Node.js (v16 or above)

## How to Run the Project

1. Download or clone this repository
2. Open the project folder in Command Prompt / Terminal
3. Run the following commands:

   npm install (one by one)
   node server.js

4. Open `index.html` in any modern web browser

## Description
This project is a community-based discussion platform where users can browse posts, explore communities, and interact using likes, comments, and sharing features.
